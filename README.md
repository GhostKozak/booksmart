## Getting Started

1. **Export** your bookmarks from your browser (Chrome, Firefox, Edge, etc.) as an HTML file.
2. **Drag & Drop** the file into BookSmart.
3. **Organize** using the dashboard and tools.
4. **Export** the cleaned list back to your browser.
